Find the Smaller Angle PrepBuddy has an analog clock which consists of two hands one for hour and another for minute. She wants to calculate the shorter angle formed between hour and minute hand at any given time. Note: You have to complete Minimal_Angle function. No need to take any input. 

    Ans:
    
    var Minimal_Angle = (h, m) => 
    {
        const ma = m * 6;
        const ha = (h * 30) + (m * 0.5);
        const angle = Math.abs(ha - ma);
        return Math.min(angle, 360 - angle);
    };

    var Reverse_Number = (N) => 
    {
      let noDigit=0,m=1,reverse=0;
      let n =N;
      while(n>0){
        noDigit++;
        n=Math.floor(n/10);
      }
     
      for(let i=1;i< noDigit;i++){
        m=m*10;
      }

      }
